# Get started

## Clone the repo

```bash
git clone https://github.com/arjansunar/introcept-internship-backend.git
```

## Install the dependencies

```bash
cd introcept-internship-backend
npm i
```

## Run the node server

```bash
# for development uses nodemon
npm run dev
```
